# Description
Rentals is a rentals advertising and searching web app where you can post, update, delete listings about room for rent, view listings, filter listings, and contact the owner of the listing. 

# Stack
- MongoDB
- Node.js
- React.js
- Express.js
- Redux
- Bootstrap

# Getting started

```
cd client
npm run start

cd server
npm run start
```
